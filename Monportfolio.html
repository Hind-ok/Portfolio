Latest Project: University Note-Taking App

This project was developed as part of a group effort using Design Thinking methodology. We utilized Figma for UI/UX design to create an intuitive and user-friendly interface. The development process followed the Scrum framework, ensuring efficient collaboration and iterative progress.

The application, built with Flutter, is designed to help university students improve their note-taking skills and adapt to higher education demands. It includes features such as organized note categorization, reminders, and study analytics.

Target Audience: University students.

Key Technologies: Flutter, Figma, Scrum.
